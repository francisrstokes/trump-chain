How it works

TrumpChain takes speech data from Trump and others, and then feeds that data to a Markov Chain, which is a memoryless state machine that transitions probabalistically from word to word based on how often those transitions came in the training data.
Basically, if you put in some Trump and some Hitler, you'll get something which is a bit like both. This can lead to some hilarious results.
TrumpChain was developed by Francis Stokes, and is open source on github.
